## class balethon.objects.*User*

**این کلاس به عنوان یک کاربر ایفای نقش میکند**

### پارامترها

- **id** (`int`)
    > آیدی کاربر

- **username** (`str`)
    >  نام کاربری کاربر

- **first_name** (`str`)
    >  نام کوچک کاربر

- **last_name** (`str`)
    >  نام خانوادگی کاربر

- **language_code** (`str`)
    >  کد زبان کاربر

- **is_bot** (`bool`)
    >  آیا کاربر بات است
