## Client.*invite_user()*

**دعوت کردن یک کاربر به یک چت**

### پارامترها

- **chat_id** (`str` | `int`)
    > آیدی چتی که یک کاربر به آن دعوت میشود

- **user_id** (`str` | `int`)
    > آیدی کاربری که دعوت میشود

### مقدار بازگشتی

> `bool`

### مثال

```python
await bot.invite_user(1234567890, 123456789)
```
