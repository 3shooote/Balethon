## class balethon.event_handlers.*RPCError*

**کلاس بیس همه ارورها با کمک این کلاس میتوان به همه انواع ارورها رسیدگی کرد**

### پارامترها

- **code** (`int`)
    > کد مخصوص ارور (اختیاری)

- **description** (`str`)
    >  توضیحات ارور (اختیاری)

- **reason** (`str`)
    > سرویسی که باعث پرتاب ارور شده (اختیاری)
